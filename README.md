## Overview

Helpers and utilities for kotlin (or jvm languages)

## Installation

Add the dependency:

```xml

<dependency>
    <groupId>com.github.sokomishalov.commons</groupId>
    <artifactId>commons-[module]</artifactId>
    <version>1.2.1</version>
</dependency>
```

Available modules now are:

- core
- logging
- serialization
- coroutines
- spring
